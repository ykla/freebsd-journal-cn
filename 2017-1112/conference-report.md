# 会议报告

会议报告  作者：Benedict Reuschling

## EuroBSDcon 2017 之旅

在法国巴黎举办的 EuroBSDcon 2017 是备受期待的盛会。整个年度里，无论大小 BSD 聚会，都能听到人们在谈论这次会议，并筹划着如何前往。甚至在会议开幕前数月，它已被宣称为迄今规模最大的一届，吸引了全球各地的参会者。对我而言，这是在 EuroBSDcon 上讲授我首场教程的机会，也是我第一次踏上法国的土地。

## 到达

周二，我飞抵巴黎，很快便与 Allan Jude 和 Niklas Zeising 一同共进晚餐——他们俩早几天就和朋友到了巴黎，毕竟这里是巴黎！当晚晚些时候，Ed Maste 加入我们，在一家印度餐厅里大家叙旧聊了许久。第二天，在董事会会议之前，Ed Maste 与我和 FreeBSD 基金会的 Deb Goodkin 在附近一家极具法式风情的小酒馆共进午餐：阳光明媚，小广场上摆着酒馆的桌椅，本地居民也在享受这一天。在了解各项项目的进展、一些文书事务、必要的财务状况以及 2018 年关于会议赞助和活动出席的规划后，多数成员回到酒店房间休息。而 Deb 陪同我利用这唯一的一点观光时间散步走向巴黎圣母院大教堂，无论内外都令人印象深刻。弥撒正在进行，歌声在数百年历史的建筑中回荡，阳光与烛光交织。当晚在一家出色的意大利餐厅与 George Neville-Neil 等人共进晚餐，席间话题从 FreeBSD 延伸到更多领域。

## 开发者峰会第一天

第二天，我们步行前往 Mozilla 办公室——一栋装饰丰富的美丽老建筑，与现代技术形成鲜明对照。Mozilla 是我们为期两天的 FreeBSD 开发者峰会的东道主。我参与组织了这次开发者峰会，制定日程，并邀请大家提交工作组提案和演讲。在 Baptiste（会议组织者之一）及其助手为所有参会者办理注册后，我们享用了早餐（当天的第二顿！），让大家在 devsummit 正式开幕前碰面交流。

我用从未有过的方式拉开了 devsummit 的序幕。谁能答对最多我准备的关于法国的问题，谁就能获得 Mogics Power Bagel。这些问题通过 Kahoot.com 呈现。Kahoot 是一个 Web 应用，参与者可以用智能手机或笔记本连接上来回答当前投射在屏幕上的问题。窍门不仅在于答对，还要比其他人更快。每一轮结束后会显示当前最高分，参与者也会在自己的设备上收到反馈。这是一种有趣且互动的方式，用来了解大家知道些什么，而这款应用在教育界非常流行。经过最初几轮，明显没有人能超越 Dag-Erling Smørgrav，他赢得了 Power Bagel 大奖。今后的活动上还会有这个游戏，敬请期待！

接下来，我们开始了工作进展讨论环节。我邀请大家就当前的工作提交 7 分钟的小型演讲。这一环节限时，目的是让大家保持专注，同时引起他们的兴趣，以便在茶歇时继续讨论。新参会者能轻松结识其他开发者，并了解他们在做什么。我们听到了关于 GELI 补丁的演讲——如果所有 provider 都使用相同的口令和密钥文件，该补丁允许附加多个 provider。Colin Percival 详细介绍了内核在启动时各阶段所耗费的时间，并提出缩短启动时间的思路，他的目标是将内核初始化时间从大约 11.5 秒压缩到 1 秒左右。Olivier Cochard-Labbe 展示了他提升 FreeBSD 数据包转发性能的工作以及其中面临的挑战，他用火焰图和图表来展示线性参考号与各种尚未提交的补丁和优化之间的关系。

午餐休息后，Deb Goodkin 介绍了 FreeBSD 基金会的最新进展，包括当前的开发项目以及在全世界推广 FreeBSD 的努力。FreeBSD 基金会的项目经理 Scott Lamons 以及来自 Intel 的 Glenn Weinberg 向我们介绍了当前与 Intel 合作的状况。讨论的主要目标是促进基金会、Intel 与 FreeBSD 社区之间的协作，并改进 FreeBSD 对 Intel 产品的支持。下午余下的时间麦克风开放。人们不时走到台前做简短即兴演讲，或就他们正在处理的问题征求意见。当天晚些时候，我们有一次安排好的聚餐，对结识和与更多参加教程或刚到的人交流非常有益。

## 开发者峰会第二天

devsummit 的第二天以 Ilya Bakulin 关于基于 CAM 的 MMC/SDIO 栈现状的演讲开场。这是一项重要的工作，因为许多嵌入式主板通过这一接口连接各种 WiFi 模块。这项工作进入代码树后，FreeBSD 便能在许多目前必须通过 USB 等外部方式接入 WiFi 的主板上访问 WiFi。接下来，core 秘书 Matthew Seaman 与其他成员举行了 core 团队问答环节。这让我们有机会倾听开发者对当前 core 团队工作的反馈。我们再次向那些今年没去过 BSDCan 的人提及 FreeBSD 社区流程，因为该流程在 BSDCan 上首次向更广泛的受众介绍过。Allan Jude 和我也回顾了首次进入 core 团队的经历，并鼓励大家在明年的选举中竞选 core。

茶歇过后，Netflix 的 Scott Long 发表了关于未来为 FreeBSD 提供更好的非一致性内存访问（NUMA）支持的演讲。少数人参与其中，与会者一致认为这是重要的开发领域。我短暂离场去迎接 Christian Schwarz。尽管他还是即将开始本科毕业论文的学生，但他已经在创建名为 zrepl 的 ZFS 复制方案上投入了大量工作。我对此印象深刻，便邀请他到 devsummit 展示他的项目，反响很好：大家不仅对他的工作印象深刻，也对他演讲的质量赞不绝口。当有人建议他明年向 AsiaBSDcon 和 BSDCan 投稿时，我露出了笑容。这正是这一活动的意义所在：社区新人可以展示自己的成果，并从开发者那里获得反馈和鼓励，进而促成更多的协作和代码贡献。

我不得不离开 devsummit 去讲授 Ansible 教程，但我听说 devsummit 余下的时间成立了两个工作组：一个关于沙箱（Capsicum、Casper 等），另一个关于 FreeBSD 的持续集成。

当我到达教程场地时，遇到了 Lucas，他当天上午讲授了“ZFS 核心概念”教程。我向 EuroBSDcon 提交了更新版的“用 Ansible 管理 BSD 系统”教程，该教程今年早些时候曾在 AsiaBSDcon 讲过。当时就有很多人对这门教程感兴趣，EuroBSDcon 上来的人更多——座无虚席！我喜欢教程的互动性，因为听众中常常有人能提供更多细节或替代方案。这让整个环节成为大家共同学习的体验，到结束时连教程讲师自己也学到了新的信息。茶歇时，大家尝试示例或索要幻灯片以便日后学习。反馈良好，我考虑把这门教程作为第三版投递到其他 BSD 会议（BSDCan 2018 是个选择）。

当晚我们来到又一家很棒且别致的餐厅，内部像酒窖，拱形屋顶。当然，会议前夜，到场的人更多了。我注意到了 Brendan Gregg，他的主题演讲让我非常期待。席间有许多故事、俏皮话和其他对话，伴随着美食和饮品，之后大家各自返回酒店。

## 会议第一天

会议包含你能从优秀 BSD 会议中期待的一切：主题演讲、三个并行分会、定时茶歇（午餐时间更长）、分组讨论室、录制团队和直播，当然还有出色的走廊交流。会议在玫瑰十字会神庙举行，空间足够容纳 300 多名参会者。周围是埃及雕像，墙上刻着象形文字——这大概就是那个时代的 ASCII。像所有 BSD 会议一样，我会提前制定想要参加的演讲日程。但到了现场，我有时会和人们进行有趣的讨论而错过演讲的开场。其他时候，我会临时决定走进截然相反的演讲——结果好坏参半。录像让我能在事后观看错过的演讲，但现场聆听好演讲是无可比拟的。

第一天，我错过了 Antoine 和 Baptiste 的“BSD 之战”演讲，因为我得监考 BSDA 考试。我在今年的 AsiaBSDcon 上看过第一部分，所以并不算完全错过，而给人机会让自己的 BSD 知识获得认证，这本身就是错过演讲的好借口。

社交活动在埃菲尔铁塔旁的一艘船上举行。这次并非传统的围桌晚餐，而是站立的晚宴。侍者整晚穿梭，奉上只有法国人才能制作的美味小食。还有开放酒吧。人们结成小群又重新分散，与许多人交谈。此外，船在移动，夜幕降临，我们看到了灯火通明的巴黎夜景。有时我们从桥下经过，桥身几乎就在头顶上方一臂之遥。有时其他船只超越我们，或我们看到人们在河岸边跳舞。这是美好的夜晚，关于 BSD 和其他话题有许多精彩的对话。

## 会议第二天

第二天和第一天一样精彩。我在 FreeBSD 基金会的展台帮忙，与人们交谈，结识新朋友。其中许多人在我得知他们名字之前就认识我，因为他们观看 BSDNow，自 Kris Moore 从麦克风前退休后我一直是主持人之一，为大家带来 BSD 世界的最新动态。我很享受这些反馈和鼓励。

不知不觉，Brendan Gregg 的闭幕主题演讲开始了。我坐在第一排，因为在闭幕环节我有任务要做。Brendan 修改了一些性能分析方法论的幻灯片，也加入了围绕 BSD 工具的新幻灯片，这样对所有人都有新内容。演讲可在此观看：<http://www.brendangregg.com/blog/2017-10-28/bsd-performance-analysis-methodologies.html>。

在闭幕环节，人们对未来几届会议做了推介，并鼓励大家参加和投稿演讲。各个 BSD 项目简短介绍了自己的工作，Deb Goodkin 和我向四位 FreeBSD 开发者致谢。

Baptiste 和 Antoine 为组织这次会议付出了大量心血，到最后嗓子都哑了，但每个人都感谢他们的辛勤付出和奉献，使这届 EuroBSDcon 如此成功。我和一小群人共进了在巴黎的最后一顿晚餐，回顾这次会议。有些人在法国多留了几天，但我必须飞回国，心里清楚这绝不是我的最后一届 EuroBSDcon。•
