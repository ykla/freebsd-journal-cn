# 会议报告

比利时以美味的啤酒、最棒的巧克力和可口的华夫饼闻名！这里也是欧洲最大的开源会议举办地。FOSDEM 2019 于 2 月 2 日至 3 日在比利时首都布鲁塞尔举行。我很幸运今年又能参会，同时参会的还有 2 月 1 日同期举办的 FreeBSD 开发者峰会。

开发者峰会有 20 多位参会者。我们就 bug 修复、黑客松、谷歌 Summer of Code 项目构想、在 Pinebook 上运行 FreeBSD 等话题展开了热烈讨论。这样与社区其他贡献者面对面交流的机会，对项目的成长和长远发展十分珍贵。与会者彼此受到启发，回家后干劲十足地投入自己的项目，或加入会议期间和晚上喝啤酒时聊到的心仪项目。今年想参加峰会的人非常多，明年我们会找一个更大的场地，让更多人能参与。

FOSDEM 第二天在布鲁塞尔自由大学开幕。FOSDEM 是由社区为社区组织的免费、非商业活动，目的是给自由和开源软件开发者及社区提供一个相聚的场所，完全免费！我记得今年有超过 8000 名参会者，走廊里非常拥挤。今年我们很幸运又拿到了展台，位置仍然紧挨着 Illumos 项目。Allan Jude 喜欢站在两个展台之间，这样就能向来访两个项目的人聊 ZFS。和往年一样，我们有大批 FreeBSD 志愿者帮忙值守展台，他们都来自欧洲各地，对 FreeBSD 充满热情。

现场有一个 BSD 开发者房间（devroom），能容纳 100 名对 BSD 感兴趣的人，他们在这里相聚并聆听了 13 场关于各 BSD 项目的演讲。我用一场《FreeBSD 的 25 年》演讲为 devroom 拉开帷幕。录像可以在这里观看：<https://fosdem.org/2019/schedule/event/25_years_of_freebsd/>。

这一天剩下的时间，我都在和到我们展台的人交流，了解到更多使用 FreeBSD 的公司，并与其他开源社区成员交谈。看到那么多人来到我们展台，我非常激动。虽然很快就发完了我们精美的行李牌和卷尺，但最终发出了 1000 张贴纸，“卖”出了 40 件 T 恤，还派发了大量《FreeBSD 入门》传单。我看到许多志愿者就内核的具体部分和功能展开深入的技术讨论。

很高兴看到 Shteryana Shopova 来到现场并重新投入 FreeBSD。她曾在保加利亚承办 EuroBSDCon 2014，并参与 FreeBSD 各部分的工作，专注网络方向。

我还听了 Jonathan Looney 在主会场做的题为《Netflix 与 FreeBSD——用开源交付流媒体视频》的演讲。这是对使用 FreeBSD 的好处的有力见证。会场座无虚席，演讲结束后人们上前向 Jonathan 提问。他的演讲可以在这里观看：<https://fosdem.org/2019/schedule/event/netflix_freebsd/>。

第二天同样疯狂、同样拥挤，不过偶尔你真的能在走廊里走动而不被肘击。

除了与更多商业用户建立联系、让人们在自己的桌面电脑上摆弄 FreeBSD 之外，FreeBSD 社区中的一些人还讨论了未来 12 个月在更多会议上设立 FreeBSD 议程和工作坊的想法。其中包括在 linux.conf.au 设立 FreeBSD 议程，在 All Things Open 和 womENcourage 举办 FreeBSD 入门工作坊，仅举几例。

总而言之，FOSDEM 是推广 FreeBSD 必须参加的会议。主会场有几场 FreeBSD 演讲，我们会鼓励更多人未来提交演讲，以提升 FreeBSD 的可见度。我们期待基金会继续在现场驻足，让公司和用户能与我们交流他们在做什么、我们能帮上什么忙；我们也会继续赞助 FreeBSD 开发者峰会，当然还会提供各种酷炫的周边。

DEB GOODKIN 是 FreeBSD 基金会的执行董事。她很高兴在基金会迎来了第 14 个年头，并为她那支勤奋敬业的团队感到骄傲。她在数据存储行业工作了 20 多年，从事工程开发、技术销售和技术市场营销。不工作的时候，你会在公路、碎石路或山地自行车道上看到她，为跑步比赛训练，和狗一起徒步，或者在各种系统上摆弄 FreeBSD。
