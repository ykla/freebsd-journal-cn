# 会议报告

作者：**Deb Goodkin**

## FOSDEM 2018 / BSD 开发者会议室 / FreeBSD 开发者峰会

我最近有机会参加了 FOSDEM 2018（<https://fosdem.org/2018/about/>），这是一个为期两天、免费的自由及开源软件会议，于 2 月 2 日至 3 日在比利时布鲁塞尔举行。它由志愿者自 2000 年起组织举办，是年度盛会，而且完全免费！正因为免费，参会人数众多——今年有超过 5000 名参与者！

虽然多年来 FreeBSD 一直在这次会议上得到推广，但这还是我第一次参加。我不太确定会遇到什么。我曾听人说现场人山人海，别指望能找到午餐，除非你愿意在午餐车前排 30 多分钟的队。事实证明我听到的多半都是真的，但我不得不说，这是一场了不起的会议，也是推广 FreeBSD 认知度的绝佳机会。

基金会赞助了会议前的 FreeBSD 开发者峰会以及会议上的 FreeBSD 展位。由于联合航空允许托运两件行李，每件 70 磅，我决定在行李箱里塞进尽可能多的 FreeBSD 期刊、宣传册和赠品。最后我得到了两个超级重的行李箱，在行李传送带旁搬运和抬下都有些困难。但是，旅行时来一场不错的锻炼又有什么不好呢？

基金会赞助这类活动是为了推广 FreeBSD，并为项目招募新的用户和贡献者。我们也在世界各地赞助开发者峰会，为 FreeBSD 贡献者提供面对面协作的机会，分享知识，并在 FreeBSD 项目的不同领域进行富有成果的讨论。

## FreeBSD 开发者峰会

峰会在当地一家酒店的地下室举行。Benedict Reuschling 组织并主持了这次活动，约有二十几位参与者。峰会上午 9 点开始，每个人做自我介绍，包括自己在做什么工作，以及当天有兴趣讨论的话题。

介绍完毕后，Allan Jude 谈到了谷歌编程之夏（GSoC）项目，说明了典型的项目形态，以及为什么我们需要更多小型项目提交，还有更多的志愿导师。我提到了 Outreachy 项目，该项目仿照 GSoC，但面向女性和少数群体。多年来我一直想参与这个项目，终于相信我们已准备好支持它。GSoC 对项目非常有益，我不愿从它那里分走资源。自这次会议之后，我们决定申请六个月后的下一期。

来自 Linux Professional Institute 的 Matthew Rice 和 Fabian Thorns 谈到他们接手 BSD 认证的事宜，解释了原因以及他们扩展在线教程的计划。目前他们提供不同级别的 Linux 认证，以及 DevOps 等其他领域的认证。展望未来，他们计划整理两套操作系统通用的资料，提供与操作系统无关的教学内容。他们有一个由 Dru Lavigne 领导的 BSD 顾问小组，由她对 BSD 认证支持进行监督。

讨论的其他话题包括 DTrace、图形支持、存储以及可能的发布模式。讨论富有成果，我感觉我们收获颇丰。许多参与者在附近一家餐厅共进晚餐继续交流。我参加了一个由谷歌赞助的开源领导者晚宴。与 FreeBSD 之外的其他开源人士交流很重要，可以了解他人在做什么以改进自己的项目和社区。这也是为维护开源这一健康生态系统发声的机会。

## FOSDEM 与 BSD 开发者会议室

正如我前面提到的，FOSDEM 规模巨大！共有 651 位演讲者、690 场活动、57 个议题 track，全部占据了布鲁塞尔自由大学（Université Libre de Bruxelles）Solbosch 校区多栋建筑中的 33 间会议室。会议设有主题演讲、主议题 track、至少 42 个开发者会议室、闪电演讲以及认证考试，包括 BSDCG 考试。

BSD 开发者会议室由 Rodrigo Osorio 组织，在 FOSDEM 第一天举行。当天共 11 场演讲，听众爆满，超过 100 人。我第一个发言，讲题是《The FreeBSD Foundation and How We Are Changing the World》（<https://fosdem.org/2018/schedule/event/the_freebsd_foundation_how_we_can_change_the_world/>）。

有少数人想了解 FreeBSD 的更多信息。这是我在准备时担心的——房间里会不会都是 BSD 圈的人？还是会有非 BSD 圈的人？我通常用这个演讲来突出我们为支持 FreeBSD 所做的工作，所以这是我的重点。在与一位想了解更多技术信息的人交谈后，我意识到明年应该再做一个 FreeBSD 简介或 FreeBSD 优势的演讲，而不仅是基金会亮点演讲。这是有价值的反馈。

演讲结束后，几位听众索要我的名片，想与我讨论他们公司使用 FreeBSD 的事。一位先生想讨论我们是否考虑提供 FreeBSD 支持，引发了一场长谈。

我听了其他几场觉得有意思的演讲，包括《BSD from scratch——from source to OS with ease on NetBSD》以及《pot: another container framework based on jails and ZFS》。

我们还有一个 FreeBSD 展位，在美国也叫桌子或摊位。FreeBSD 贡献者志愿值守展位并与大家交流。我在午餐时间赶到我们被挤爆的展位。我立刻明白了为什么今年这么多人问我是否有展位。展位前的人多得都看不到我们的桌子了！大家一致同意明年我们需要一个大标志来让自己脱颖而出（不是双关！）。然而，这并没有阻止成百上千乃至数千人驻足我们的展位。访客中大多数对 FreeBSD 较为熟悉。许多人有关于自己配置的问题，我们的志愿者都能帮助解答。我把树莓派 3 FreeBSD 演示环境放在展位上，吸引了许多人前来查看。我们还做了一块牌子，展示了许多使用 FreeBSD 的公司。目的是突出 FreeBSD 正在成长，有许多知名公司成功使用它。最后，还有几位真心想为项目做贡献的人，这令人振奋。

我们的展位紧邻 illumos 展位，这让 Allan Jude 能坐在两个项目之间，在 illumos 上用 UEFI 安装 bhyve。

我当天剩下的时间不是在展位后面就是在展位前面。有时我们有五个人同时值守。

贴纸是那里的硬通货。你必须有贴纸！有一次我们以为发完了，幸好我又找到一些藏起来的。我们赠送的行李标签也很快被拿光，有人建议我们为它们写一个 man page，因为我们在文档方面特别在行。我觉得这是个绝佳主意。明年，我们一定要带至少 1000 张贴纸，也许还可以卖点 T 恤。

免费赠品、Groff the BSD Goat（BSD 山羊吉祥物）以及树莓派 3 演示吸引了许多人来到展位，但大多数人留下来与我们交谈，要么想多了解 FreeBSD，要么如果他们已熟悉 FreeBSD，就问些问题。

## FOSDEM 第二天

FOSDEM 第二天和第一天差不多，不同之处是我们的展位有更多志愿者。此外，你可以在人群中穿行而不用肘击别人。我们还发现，如果去得够早，午餐车排队其实很短！除了与参会者聊 FreeBSD，我也花时间认识了一些社区成员，其中一些我在社交媒体上交流过，另一些我从未见过。我不仅喜欢听大家在做什么，这些交流也提供了机会，让我们讨论如何提供帮助，鼓励他们在会议或聚会上演讲，或就自己工作撰写文章。与社区成员面对面交流能促成大量信息交换。

当天晚些时候，又出现了一场有趣的讨论：一位 FreeBSD 开发者评论说，他不希望人们仅仅因为 systemd 才切换到 FreeBSD，而是因为 FreeBSD 诸多积极特性。我问他最喜欢 FreeBSD 的三件事。随后项目中的另外两人也加入讨论，使得这场对话信息量丰富！

部分原因如下：

- DTrace、ZFS、netgraph 和 DummyNet
- 基本系统与软件包之间的接口定义清晰
- 基本系统是一个完整的系统，包含内核、用户空间和工具
- 海量第三方软件可在 Ports 中获取

会议结束后，大家都主动帮忙收拾。规则是我们要把场地恢复成最初的样子。整个过程花了一些时间，不是因为要打包和清理的东西很多，而是因为我们需要和这一周末遇到的许多人道别。

## 巧克力任务

回程和我一起吃饭的人不知道的是，我还有一项任务要完成：我得带巧克力回去给朋友和家人。不巧的是，那天是周日晚。我以为不会有店开门，但至少想试试。我们尽快离开，七个人叫了两辆 Uber。我先下单时还有点得意。两组人在繁忙的路边等待，然后我这组在追踪器上看到我们的 Uber 一动不动。当 Benedict 那组被接走并笑着向我们挥手时，我们的 Uber 还没动，我们继续站在刺骨的寒冷中困惑不已。幸好那辆 Uber 被取消，我很快又订了一辆，不久就到了。

我们在酒店放下行李后很快汇合，出发去找巧克力。在大广场（Grand-Place）附近找巧克力店，就像在博尔德找健康食品店一样——遍地都是！对我们来说幸运的是，它们都开门营业。最棒的是你可以在所有店里试吃巧克力，然后再决定最喜欢哪家店（或哪种巧克力）。我们最后在 Allan Jude 最喜欢的店 Corné Port-Royal 买了大量巧克力。这是社区一起玩乐的又一次社交机会。

比利时巧克力再多也不嫌多！

我们的社区由一群真正出色的人组成。他们喜欢彼此相处，无论是共同参与项目，还是结伴探索新城市。我们这一行七位 FreeBSD 贡献者在最后一晚外出就餐时展现了真性情。我们满足于巧克力采购之后，迅速把战利品送回酒店，然后出发去吃饭。Benedict 之前找到一家他认为看起来不错的餐厅，我们就去那里看看。不幸的是，我们很快意识到有问题：我们中有一位是纯素食者，两位是素食者。这家店没有纯素选项。虽然我们这位 FreeBSD 纯素朋友表示他可以自己去吃，但我们绝不会丢下他。

决心找到一家适合纯素者的餐厅，我们跟随谷歌地图出发。可惜进展不太顺利。我们找到的一家店关门了。我们没有放弃，我走到隔壁，恰好是一家巧克力店。我努力克制住自己，问店员她是否知道附近有纯素餐厅。幸运的是，她推荐了仅几条街外的一家。找到那家店不容易，等我们终于找到时，他们没有八个人的座位。我有没有提到还有一位 illumos 的人加入了我们的搜寻？是的，我们是一群友好的人！于是我们决定分头行动。我们三个人留在了适合纯素者的咖啡馆，其余的人去找到了一家他们满意的餐厅。我们的纯素朋友享用了他的纯素蛋糕！我也喜欢在那里的餐食，没错，是纯素的。

总体而言，这是一次很棒的经历。它不仅提供了推广 FreeBSD 的绝佳机会，也让我们能结识社区中的其他人，以及对开源感兴趣的人。我期待今年在世界各地的更多开源会议上推广 FreeBSD。•
