# FreeBSD 基金会报告

2020 年已至，我想借此机会与期刊读者分享：2019 年对支持 FreeBSD 项目与社区而言是多么精彩的一年！为什么这么说？因为回顾过去一年，我意识到我们在全球参加了多少活动，又以多少种方式影响了人们的生活。从推广 FreeBSD 到实现 FreeBSD 功能，我的团队始终在帮助 FreeBSD 成为最优秀的开源项目与操作系统。

2019 年，我们专注于支持项目最需要帮助的几个关键领域。第一个领域是软件开发。无论是签约 FreeBSD 开发者从事 WiFi 支持等项目，还是调配内部员工快速实现硬件规避方案，我们都已介入，帮助 FreeBSD 保持创新、安全与可靠。

软件开发还包括支持让开发流程顺畅运转的工具与基础设施。我们安排团队成员主导持续集成工作，并积极参与 clusteradmin 与安全团队。事实上，去年我们提交了超过 1200 处代码变更；评审并反馈了数百处其他开发者的变更，还将持续集成测试通过率提升了近 100%！详情可阅读我们的软件开发项目与 CI 及测试进展。

我们的推广工作聚焦于为项目招募新用户与贡献者。我们在 21 个国家参加并参与了 38 场会议与活动。从发表 FreeBSD 演讲、举办工作坊到展台值守，我们与数千名参会者进行了面对面交流。你可以在此查看我们推广工作的完整回顾。

我们的差旅也提供了直接与 FreeBSD 商业用户、个人用户、贡献者以及未来的 FreeBSD 用户/贡献者交流的机会。我们看到这些组织与个人对 FreeBSD 的使用与兴趣都在增长。这些会面让我们更了解组织需要什么，以及他们与其他个人正在做什么。这些信息有助于指导我们应资助哪些工作。

去年 FreeBSD 期刊也迎来了重大变化。基金会决定将其转为免费出版物。如前所述，FreeBSD 基金会使命的重要部分是在全球提升 FreeBSD 的知名度。FreeBSD 期刊是让关注 FreeBSD 的人士了解最新动态的优秀平台，文章及时且信息丰富。因此，基金会判定，向所有人免费开放所带来的收益超过其制作成本。

2019 年，你的捐赠帮助我们继续支持 FreeBSD 的关键领域，例如：

- 操作系统改进：提供人员立即响应紧急问题，实现新功能与特性，保障你赖以依靠的创新与稳定。
- 改进并扩大测试覆盖、持续集成与自动化测试，配备专职软件工程师，确保你获得最高质量、安全且可靠的操作系统。
- 安全：提供工程资源，增强安全团队的能力与响应速度，让你在安全事件发生时无后顾之忧。

作者：**DEB GOODKIN**

> "我们已介入，帮助 FreeBSD 保持创新、安全与可靠。"

- 通过全球 FreeBSD 推广与倡导工作扩大 FreeBSD 贡献者与用户规模，包括向中国、印度、非洲与新加坡等地区扩展。
- 在全球更多会议、聚会与大学举办 FreeBSD 工作坊与演讲。
- 提供开发者峰会、供应商峰会与企业参访等机会，促进商业用户与 FreeBSD 开发者之间的协作，并帮助变更合入 FreeBSD 源码树，构建更庞大、更健康的生态。
- 为所有希望了解 FreeBSD 最新进展的人士提供专业出版的 FreeBSD 期刊。

这一年我们成果颇丰，但我们仍只是一个 501(c)(3) 小型组织，专注于支持 FreeBSD，而非像其他许多开源基金会那样的行业组织。

我们要继续并扩大对 FreeBSD 的投入，完全依赖你的捐赠。我们需要你的帮助来延续并扩大正在进行的工作。你的支持直接影响项目。

如果你和我们一样热爱 FreeBSD，请帮助我们传播 FreeBSD，并且如果你尚未捐赠，请今天就捐一笔。携手并肩，我们能为 FreeBSD 取得更多成就。
